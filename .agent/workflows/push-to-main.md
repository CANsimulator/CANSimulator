---

This workflow automates the process of committing and pushing your local changes to the remote `main` branch.

# Commands

1. **Verify the current branch**
// turbo
`git branch --show-current`

2. **Stage all changes**
// turbo
`git add .`

3. **Commit changes**
// turbo
`git commit -m "feat: update components and simulator logic"`

4. **Push to main**
// turbo
`git push origin main`

5. **Final Status**
// turbo
`git status`
